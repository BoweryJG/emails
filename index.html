💡 The Spark

Creating a Tactile Language

It all began with a simple thought: if phones can vibrate, maybe we can use those vibrations — different patterns, rhythms, strengths — to help guide people by touch. Almost like creating a subtle new "language" that you can feel instead of see or hear.

vibration_patterns.js
// Defining vibration patterns for navigation
const navigationPatterns = {
  "turnLeft": ["short", "pause", "short"],
  "turnRight": ["long", "pause", "short"],
  "obstacle": ["short", "short", "short", "long"],
  "destination": ["long", "long", "pause", "long"]
};

function triggerPattern(patternName, intensity = "medium") {
  const pattern = navigationPatterns[patternName];
  const timings = translatePattern(pattern, intensity);
  navigator.vibrate(timings);
  // Also trigger spatial audio cue with directional feedback
  playSpatialAudio(patternName, {
    direction: currentHeading,
    distance: nextWaypointDistance,
    intensity: calculateIntensity(nextWaypointDistance)
  });
}
🧠 The Big Idea

That led me to thinking even bigger: what if we could build a system where visually impaired people could navigate safely and independently using just their smartphone — combining vibration patterns and spatial audio (where the sound comes from different directions) to intuitively guide them?

No expensive new device, no complicated equipment — just using what they already have in their pocket.

The more I thought about it, the more it clicked. We could teach the brain to recognize certain vibration "signals" — like a short buzz for left, a longer one for right, or a special pattern for obstacles ahead — while stereo sound quietly nudges you in the right direction.

Building on Microsoft's Foundation

And here's the craziest part: Microsoft had already created something similar a few years ago called Soundscape, which used 3D sound to help blind users navigate cities. But last year, they decided to open-source the entire project — which means the entire foundation is freely available to build on now.

It's a huge head start if I decide to pursue this.
